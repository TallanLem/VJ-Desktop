# VJ-Desktop

Стартовый прототип desktop-приложения подбора насосов на PyQt5.

## Что уже реализовано
- Центральный `main.py` со всеми ключевыми переменными настройки в начале файла.
- Синхронизация публичных Google Sheets (CSV export по `gid`) и загрузка в SQLite.
- Главное окно в плиточном стиле с:
  - поиском по артикулу/наименованию (от 3 символов);
  - левой областью с вкладками: график/чертеж;
  - правой фиксированной панелью рабочей точки, параллельной работы и списка параметров;
  - кнопкой генерации тех. листа.
- Построение 2 графических полей (`Q-H` + `Q-КПД`, `Q-P` + `Q-NPSH`) с полиномами,
  поддержкой параллельной работы и кривой трубопровода.

## Запуск
```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
python main.py
```


## Обновления по замечаниям
- Исправлена верстка заголовков `QGroupBox` ("Рабочая точка" и "Параллельная работа") — заголовки не наезжают на поля.
- Поиск сделан регистронезависимым по полям `Наименование` и `Артикул`, с показом выпадающего списка при вводе от 4 символов.
- Синхронизация Google Sheets теперь с детектом изменений по SHA-256 хэшу CSV; если лист не изменился, база не перезаписывается.
